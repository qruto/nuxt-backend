Use `nuxt-backend/convex/test` to register the packaged component when testing your Convex functions with [`convex-test`](https://docs.convex.dev/testing/convex-test).

```ts
import { convexTest } from 'convex-test'
import backendTest from 'nuxt-backend/convex/test'
import schema from './schema'

const t = convexTest(schema, modules)
backendTest.register(t)
```

`register(t, name?)` mounts the packaged component (default name `'backend'`) into the test harness so functions that read auth tables, send email, or call `components.backend.*` resolve correctly under test.

```ts
function register(
  t: TestConvex<SchemaDefinition>,
  name?: string, // default 'backend'
): void
// default export: { register, modules }
```
