`setupMigrations(components.migrations, { schema? })` returns `{ migrations, run }`. Define migrations with `migrations.define({ table, migrateOne })`.

```ts [backend/migrations.ts]
import { setupMigrations } from 'nuxt-backend/convex/migrations'
import { components } from './_generated/api'
import schema from './schema'

export const { migrations, run } = setupMigrations(components.migrations, { schema })

export const setDefaultStatus = migrations.define({
  table: 'messages',
  migrateOne: async (ctx, doc) => {
    if (doc.status === undefined) {
      await ctx.db.patch(doc._id, { status: 'sent' })
    }
  },
})
```

Run a migration from the CLI:

```bash
npx convex run migrations:run '{ "fn": "migrations:setDefaultStatus" }'
```
